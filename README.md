# CUA-Lark 飞书桌面自动化系统

让 AI 替你操作飞书！通过自然语言描述任务，系统会自动截图、分析并执行桌面操作。

---

## 🌟 功能特性

### v3.1 最新特性
- 🎯 **快捷连招** - `click_paste_enter` 操作，一步完成点击输入框、粘贴、回车
- 🖱️ **智能滚轮** - 支持先移动到指定位置再滚动，首次6格，错过后3格调整
- 🛡️ **稳定JSON** - 增强JSON格式规范，减少解析错误
- ✏️ **输入优化** - 修复窗口激活逻辑，避免打断用户输入
- ⏱️ **体验提升** - 初始化从5秒改为3秒，截图后2秒缓冲等待页面加载
- 🪟 **无窗口启动** - 支持 pythonw.exe 和 bat 文件一键启动
- 🛣️ **坐标归一化** - AI只需要返回0-1比例，系统自动计算最终坐标
- ⏹️ **即时终止** - 在关键节点检测停止信号，点击终止按钮后快速响应

### v3.0 特性
- 🎨 **图形界面** - `gui_app.py`，窗口固定在右上角，实时显示AI思考
- 💬 **连续对话** - 支持连续任务，AI会记得之前的对话内容
- 🗂️ **路径打开** - `open_folder` 操作，通过路径直接打开文件夹
- 💭 **纯聊天模式** - 不想操作电脑时，直接和AI聊天即可，任务自动结束
- 📘 **飞书专属优化** - 完整的飞书界面说明，让AI更懂飞书

### 核心功能
- 🤖 **AI 智能控制** - 基于豆包 2.0 Pro（火山方舟），通过视觉理解屏幕内容
- 🖱️ **完整鼠标操作** - 点击、双击、拖拽、悬停、框选、滚轮
- ⌨️ **全功能键盘** - 文本输入（支持中文）、按键、热键、密码输入
- 🔄 **闭环执行** - 截图 → AI 决策 → 执行操作 → 再截图检查，循环直至任务完成
- 🛡️ **安全机制** - 故障安全、最大迭代次数限制
- 📷 **自动截图** - 每次迭代自动保存屏幕截图

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置 API

编辑 `config.py`，填写你的火山方舟 API 信息：

```python
AI_CONFIG = {
    'api_endpoint': 'https://ark.cn-beijing.volces.com/api/v3/chat/completions',
    'api_key': 'ark-你的API密钥',      # 替换为你的密钥
    'endpoint_id': 'ep-你的接入点ID',   # 替换为你的接入点ID
    'temperature': 0.1,
    'max_tokens': 2000,
}
```

⚠️ **安全提示**：请妥善保管 `config.py` 中的 API Key，不要提交到公开仓库！

### 3. 运行程序

**方式1：图形界面模式**

```bash
python gui_app.py
```

- 窗口会自动出现在右上角

**方式2：双击启动**

- 📝 `启动CUA-Lark.bat` - 带控制台窗口（方便调试查看日志）
- ✨ `启动CUA-Lark(无窗口).bat` - 无控制台窗口（推荐日常使用）

直接双击对应文件即可启动！

**方式3：创建桌面快捷方式**

运行 `setup_shortcut.py` 自动创建桌面快捷方式（需要安装pywin32）：
```bash
pip install pywin32
python setup_shortcut.py
```
创建的快捷方式会自动使用无控制台模式。

**方式4：终端模式**

```bash
python feishu_cua.py
```

### 4. 输入任务示例

**飞书操作：**
```
打开飞书，在云文档中创建一个新文档
```

**纯聊天：**
```
你好，给我讲个笑话
```

**连续任务：**
```
任务1：在桌面创建一个新文件夹
任务2：把刚才的文件夹重命名为"测试"
```

## ⚡ v3.1 新增功能说明

### 🎯 快捷连招 - click_paste_enter
AI可以一步完成：点击输入框 → 粘贴剪贴板内容 → 按回车发送！
```json
{
  "action": "click_paste_enter",
  "x": 0.500,
  "y": 0.600
}
```
适用于发送消息、提交表单等场景，效率提升3倍！

### 🖱️ 智能滚轮策略
- **首次尝试**：直接滚动6格（足够大的步长）
- **错过调整**：如果错过目标，再3格3格返回调整
- **指定位置**：可以先通过x,y移动到特定区域再滚动
```json
{
  "action": "scroll",
  "clicks": 6,
  "x": 0.800,
  "y": 0.500
}
```

### 🛣️ 坐标归一化
AI只需要返回0-1的比例，系统自动计算最终坐标：
- x比例 × 屏幕宽度 = 最终x坐标
- y比例 × 屏幕高度 = 最终y坐标
- 例如：屏幕中间是 (0.500, 0.500)

## 📋 AI 支持的操作

AI 可以通过以下 JSON 指令控制你的电脑：

### 鼠标操作

```json
{"action":"left_click","x":500,"y":300}           // 左键点击
{"action":"right_click","x":500,"y":300}          // 右键点击
{"action":"double_click","x":500,"y":300}         // 双击
{"action":"drag","start_x":100,"start_y":100,"end_x":300,"end_y":300}  // 拖拽
{"action":"hover","x":500,"y":300}               // 悬停
{"action":"select_area","start_x":100,"start_y":100,"end_x":300,"end_y":300}  // 框选
{"action":"scroll","clicks":10}                   // 滚轮（正数向上，负数向下）
```

### 文件操作（v3.0 新增）

```json
{"action":"open_folder","path":"C:\\Users\\XXX\\Desktop\\文件夹名"}  // 通过路径打开文件夹
```

### 键盘操作

```json
{"action":"type_text","text":"要输入的内容"}       // 输入文本
{"action":"press_key","key":"enter"}              // 按键
{"action":"hotkey","keys":["ctrl","v"]}           // 热键
```

### 密码输入

```json
{"action":"type_password","password":"xxx"}       // 密码输入
```

### 任务控制

```json
{"action":"task_completed","result":"成功！"}     // 任务完成
{"action":"continue"}                             // 继续执行
```

## 🔧 各模块说明

### config.py - 配置文件

包含 AI API 配置和系统提示词。

- `AI_CONFIG`：火山方舟 API 配置
- `SYSTEM_PROMPT`：告诉 AI 我们有哪些能力，以及如何返回 JSON 指令

### screenshot.py - 截图模块

```python
from screenshot import capture_fullscreen
path = capture_fullscreen()  # 返回截图路径
```

### ai_communicator.py - AI 通信模块

```python
from ai_communicator import send_to_ai
command = send_to_ai("用户任务", "screenshot.png", screen_w, screen_h)
```

### command_parser.py - 命令解析器

解析 AI 返回的 JSON 并执行相应操作。

AI 会根据截图和屏幕尺寸估算坐标，直接使用即可。

### desktop_automation.py - 桌面操作库

核心自动化操作类，提供所有鼠标键盘操作。

### feishu_cua.py - 终端模式主程序

整合所有模块，提供终端交互方式。

### gui_app.py - 图形界面模式（推荐）

提供图形界面，实时显示AI思考，支持连续对话。

## 🛡️ 安全机制

### 故障安全

- 默认启用 `fail_safe=True`
- 鼠标移动到屏幕左上角会紧急停止
- 如果程序失控，立即把鼠标移到左上角！

### 最大迭代次数

- 默认最多执行 15 次操作
- 防止无限循环

## ❓ 常见问题

**Q: 如何知道坐标是多少？**
A: 可以先用 `pyautogui.displayMousePosition()` 查看，或者让 AI 根据截图估算。

**Q: AI 一直犯错怎么办？**
A: 可以更详细地描述任务，或者调整提示词。

**Q: 程序跑起来之后怎么停止？**
A: 鼠标快速移到屏幕左上角即可。

**Q: 可以连续给任务吗？**
A: 可以！使用图形界面模式，任务会自动保留对话历史。
